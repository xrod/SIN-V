# SIN-V

### Single-zone external-field blazar model in the Spectra of IceCube Neutrinos (SIN) V paper ([Rodrigues et al, *Astron.Astrophys.* 689 (2024) A147](https://www.aanda.org/articles/aa/full_html/2024/09/aa50592-24/aa50592-24.html)), applied here to source TXS 0506+056 as an example. For the other 31 sources modeled in the paper, see parameters in Table E.1.

### When using this model, please cite [the above paper](https://www.aanda.org/articles/aa/full_html/2024/09/aa50592-24/aa50592-24.html) as well as [AM3](https://am3.readthedocs.io/en/latest/#citing-am3).
